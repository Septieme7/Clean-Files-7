Installer 7‑Zip (Windows)

Si le serveur ou ton système local indique que 7‑Zip est manquant, suis ces étapes :

1) Télécharge 7‑Zip

Page officielle : https://www.7-zip.org/

2) Choisis la version pour Windows

Télécharge la version x64 si ton système est 64‑bits (généralement le cas).

3) Installe 7‑Zip

  1. Lance le fichier .exe téléchargé et suis les instructions d'installation.
  2. Note le dossier d'installation (ex. C:\\Program Files\\7-Zip\\).

4) (Serveur Windows) Ajoute 7‑Zip au PATH

Si ton serveur tourne sous Windows, ajoute le dossier 7‑Zip au PATH système :

setx PATH "%PATH%;C:\\Program Files\\7-Zip\\"

Ensuite redémarre le service/terminal qui exécute le serveur pour que le changement soit pris en compte.

5) Vérifie que la commande 7z fonctionne

7z --help

Tu dois voir l'aide de 7‑Zip. Si oui, le serveur pourra exécuter la commande.

Besoin d'aide ?

Si tu veux, copie-colle ici la sortie de la commande ci‑dessus dans la console, je t'aiderai à corriger.
